# vagrant-xenial64-php7.1
*Version 1.0-beta*

**Installation**  

    $ git clone https://github.com/johninamio/vagrant-xenial64-php7.1.git
    $ cd vagrant-xenial64-php7.1/ && vagrant up

**System**  

- Ubuntu 16.04
- Apache 2.4
- MariaDB 10
- PHP 7.1
- phpmyadmin

**Utilities**  

- Composer
- GIT
- NodeJS, NPM
- Bower, Grunt, Gulp
